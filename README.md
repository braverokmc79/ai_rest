


### <타입>: <변경 요약>

| 타입          | 설명                             |
| ----------- | ------------------------------ |
| `feat:`     | 새로운 기능                         |
| `fix:`      | 버그 수정                          |
| `docs:`     | 문서 변경                          |
| `style:`    | 코드 포맷팅, 세미콜론 누락 등 기능에 영향 없는 변경 |
| `refactor:` | 코드 리팩토링 (기능 변화 없음)             |
| `test:`     | 테스트 관련 코드 추가/수정                |
| `chore:`    | 기타 작업 (빌드, 의존성 등)              |
